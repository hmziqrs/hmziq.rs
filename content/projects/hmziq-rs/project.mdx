A personal portfolio website built for speed and flexibility.

**Key features:**
- Next.js and React for server-side rendering and flexibility
- Tailwind CSS and shadcn/ui for clean, responsive design
- Framer Motion and Three.js for animations and 3D visuals
- Rust with WebAssembly for performance-heavy computations
- Optimized, containerized, and easy to maintain
