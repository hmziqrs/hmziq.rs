A tool to run multiple Claude Code instances with different LLM providers — each with its own config, plugins, and MCP servers.

**Supported providers:** DeepSeek, MiniMax, Z.ai GLM, Xiaomi Mimo

**Key features:**
- Run isolated instances like `claude-glm` or `claude-deepseek`
- Wraps the stock `claude` binary for each provider
- Terminal UI for managing configurations
- Published on npm
- Runs on Node.js, Bun, pnpm, and Deno
