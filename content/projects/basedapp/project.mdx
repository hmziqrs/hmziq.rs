Built the initial version of BasedApp, a crypto wallet app for SHA2 Labs using Flutter, then pivoted to React Native for better Web3 library availability in the JavaScript ecosystem.

**Key contributions:**

- Integrated Web3 capabilities including SafeWallet, Magic Link authentication, and various blockchain functionalities
- Ported all web app functionalities including wallet creation, KYC verification, crypto wallet management, and virtual card management
- Collaborated with the designer to create a design language for the app
- Built an internal Tailwind CSS-like styling library following React Native standards
- Contributed to a Next.js-based web app with a serverless function for swap tokens breakdown
