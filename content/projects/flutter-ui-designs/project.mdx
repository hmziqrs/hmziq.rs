A comprehensive collection of beautifully crafted Flutter UI implementations supporting web, mobile, and desktop platforms.

## Features

- Cross-platform support (Web, Android, iOS, Windows, macOS, Linux)
- Responsive layouts that adapt to different screen sizes
- Complex animations and transitions
- Internationalization (English, Chinese, Arabic)
- Dark/Light theme support
- Firebase integration for analytics and crash reporting
- State management with Provider pattern
- Clean and modular architecture

## UI Implementations

1. **Healthy Food Delivery** — Food listing with parallax effects, nutritional info, rating system
2. **Hot Air Balloon** — Booking interface, flight details, interactive balloon animations
3. **Sky View** — Planet/Star viewing interface, parallax star field effects
4. **Asics Shoes Concept** — Product showcase, color variants, size selection
5. **Egg Timer Concept** — Circular timer interface, gesture-based controls
6. **Chef Craft** — Recipe browsing, cooking instructions, ingredient lists

## Key Contributions

- Built custom internationalization library from scratch
- Custom GitHub Actions script for multi-platform builds (Windows, Linux, macOS, Android)
- Custom Dart CLI script for automated translation of JSON files
- E2E tests with screenshots in both dark and light modes
