A collection of UI designs built with Flutter that runs on any platform: mobile, web, and desktop.

**Key contributions:**
- Structured code with provider state management for maximum performance
- Integrated dark and light mode support
- Built a custom internationalization library from scratch
- Wrote a custom GitHub Actions script that builds Windows, Linux, macOS, and Android apps and uploads to GitHub releases
- Wrote a custom Dart CLI script that automates translation of English JSON files into multiple languages
- Wrote E2E tests that take screenshots in both dark and light modes
- Complex gesture-based parallax animations with responsive layout
- Offline-first architecture
