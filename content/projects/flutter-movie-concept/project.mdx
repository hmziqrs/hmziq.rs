A true cross-platform UI design featuring complex scroll-based animations powered by Flutter, running on mobile, desktop, and web.

Originally designed by [Aneesh](https://dribbble.com/aneeshravi).

## Key Contributions

- Structured code with Provider state management for maximum performance
- Dark and light mode support
- Custom internationalization library from scratch
- Custom GitHub Actions script for multi-platform builds (Windows, Linux, macOS, Android)
- Custom Dart CLI script for automated translation of English JSON files
- E2E tests with screenshots in both dark and light mode
- Complex gesture-based parallax animations
