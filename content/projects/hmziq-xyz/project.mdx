A portfolio website built with Dioxus, a Rust-based web framework featuring both server and static assets.

**Key features:**
- Built as a Rust/Dioxus web app with Dioxus manifest and Cargo configuration
- Tailwind CSS for styling
- Contains HTML, Rust, CSS, and small JS/TS pieces
- Deployment/config files for Firebase
- Lightweight, component-focused structure optimized for experiments and fast iteration
