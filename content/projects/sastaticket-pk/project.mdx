Pakistan's top-ranked airline travel application, architected and developed using Flutter.

**Key contributions:**
- Multi-payment gateway integration (Easypaisa, Jazzcash, Paypro, 3D-secured cards)
- Comprehensive analytics suite (Webengage, Mixpanel, AppsFlyer)
- Automated CI/CD pipeline via GitHub Actions and Node.js at zero cost
- Google Drive API integration
- Firebase analytics and notifications
