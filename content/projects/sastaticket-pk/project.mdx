Pakistan's top-ranked and trending travel and tourism mobile application, architected and developed using Flutter.

**Key contributions:**
- Architected the code structure with the Bloc design pattern
- Configured application flavors for QA, stage, development, and production environments
- Integrated WebEngage, Mixpanel, AppsFlyer, Facebook, and Firebase analytics
- Integrated multiple local and international payment providers and 3D-secure card payments
- Developed scripts for automating intl JSON generation, boilerplate code, assets generation, and pre-deployment setup
- Built a custom CI/CD program using GitHub Actions and Node.js to build, upload APK to Google Drive, and auto-share links via email
- Wrote Dart wrappers for Kotlin and Swift-based native SDKs
- Interviewed and assessed junior and senior developers; played a pivotal role in hiring decisions
