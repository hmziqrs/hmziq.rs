A location-based and lifestyle platform spanning backend, web, and mobile — developed at FetchSky.

**Key contributions:**
- Helped develop a Node.js and MySQL back-end with CRUD API calls
- Built a custom lightweight image cropping and sizing module with NodeCanvas
- Rendered a React-based analytics dashboard on the server side, took PDF snapshots via Puppeteer, and auto-sent reports at weekly intervals
- Contributed to React web and React Native mobile applications with reusable, responsive UI components
- Integrated API calls with Redux, Immutable.js, and Redux-Saga
- Integrated Firebase analytics, Facebook Pixel, OneSignal push notifications, and deep linking
- Built custom React components with styled-components for the internal design library
- AngularJS admin panel for internal management
