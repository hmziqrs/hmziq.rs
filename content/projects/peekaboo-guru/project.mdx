A location-based and lifestyle platform spanning backend, web, and mobile.

**Key contributions:**
- NodeJS/MySQL backend architecture
- ReactJS web application
- React Native mobile app with Redux state management
- Custom image cropping module using node-canvas
- Internal admin tools: AngularJS admin panel
- Server-side rendered analytics dashboard with PDF reporting
- Puppeteer-based automation
- AWS infrastructure (S3, SES)
